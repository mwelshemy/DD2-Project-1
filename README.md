## Project Overview
This repository contains the deliverables for Project 1: Standard Cell Library Design and Characterization. We designed and characterized a 13-cell library using the SKY130 process node to generate Non-Linear Delay Model (NLDM) tables.

## Repository Contents
* `cells.spice`: A single SPICE library file containing the `.subckt` definitions for all 13 cells.
* `characterize.py`: The unified Python/OpenSPICE automation script used to simulate the 13 cells across 7 input transitions and 7 load capacitances (637 total simulations).
* `plot.py`: The script used to generate the Delay vs. Load visualization for the inverter family.
* `Project1_Report.pdf`: The final characterization report containing the analytical RC comparison, plots, and NLDM tables.
* `nldm_tables.csv`: The raw output matrices generated by the automation script.

## How to Run the Characterization
1. Ensure `ngspice` and `python3` are installed on your system.
2. Verify the `.lib` path in `characterize.py` matches your local SKY130 PDK installation path.
3. Run the script:
   ```bash
   python3 characterize.py
